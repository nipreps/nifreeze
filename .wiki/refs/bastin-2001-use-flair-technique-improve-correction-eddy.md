# Mark E. Bastin (2001)

*On the use of the FLAIR technique to improve the correction of eddy current induced artefacts in MR diffusion tensor imaging.* — Magnetic Resonance Imaging.

## Claim

The results indicate that although calibration data obtained from brain or phantom images can significantly reduce eddy current induced distortions, the optimum image realignment achievable using post-processing methods is likely to be that obtained by direct image warping techniques.
