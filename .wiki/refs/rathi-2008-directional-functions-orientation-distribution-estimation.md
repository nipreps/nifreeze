# Y. Rathi, O. Michailovich, S. Bouix, M. Shenton (2008)

*Directional Functions for Orientation Distribution Estimation* — IEEE International Symposium on Biomedical Imaging.

## Claim

This work proposes a novel function that has the ability to compactly represent signals measured using high angular resolution diffusion imaging (HARDI) and forms a directional function for which the ODF is the von Mises-Fisher (vMF) distribution.
