# T. Minka, Rosalind W. Picard (2017)

*Learning How to Learn is Learning With Point Sets*.

## Claim

This paper presents an alternative interpretation: learning how to learn is ordinary learning but on point sets, rather than points, for Bayesian models of ordinary learning.
